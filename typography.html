Bootstrap 5 Default Settings:

  • Bootstrap 5 има default font-size вредност 1rem (16px), а line-height е 1.5.

  • <p> елементите имаат margin-top: 0 и margin-bottom: 1rem (16px по default)

  • <h1> - <h6>

  • Bootstrap ги стилува HTML headings така тие се подебели и со поголем фонт.


Големина на Bootstrap headings:

h1 Bootstrap heading (2.5rem = 40px)

h2 Bootstrap heading (2rem = 32px)

h3 Bootstrap heading (1.75rem = 28px)

h4 Bootstrap heading (1.5rem = 24px)

h5 Bootstrap heading (1.25rem = 20px)

h6 Bootstrap heading (1rem = 16px)

Листа на дефиниции

Користи го елементот <dl> за да креираш листа на дефиниции:

HTML
Јазик за означување кој се користи за креирање на веб страници.
CSS
Јазик кој се користи за стилизирање на HTML документи.

Инлајн код

Користи го елементот <code> за да прикажеш краток инлајн код. Пример: <p> е параграф елемент.


Користење на скратеници преку тастатура

Користи го елементот <kbd> за да прикажеш скратеница од тастатура. Пример: Притисни Ctrl + S за да зачуваш документ.


Код составен од повеќе линии

Користи го елементот <pre> за да прикажеш претходно форматиран текст со задржување на нови редови и празни места:

    Пример за претходно форматиран текст.
    Ова е нов ред.

Заклучок

Bootstrap 5 нуди опсежен сет на класи за типографија кои помагаат во организирање и унапредување на текстот на веб страниците. Со правилно користење на заглавја, параграфи и инлајн елементи, можете да создадете читлив и естетски убав дизајн. Важно е да се внимава на контекстот и стилот на текстот, за да се подобри искуството на корисниците и да се пренесат пораките на ефективен начин.
